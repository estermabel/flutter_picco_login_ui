## Deployment

To deploy this project run

```bash
  flutter run
```

  
## Run Locally

Clone the project

```bash
  git clone https://github.com/estermabel/flutter_picco_login_ui.git
```

Go to the project directory

```bash
  cd flutter_picco_login_ui
```

Install dependencies

```bash
  flutter pub get
```

Run the project

```bash
  flutter run
```
